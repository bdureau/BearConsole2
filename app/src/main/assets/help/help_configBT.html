Bluetooth module configuration

In order to your bluetooth module to connect your altimeter to your phone you will first need to configure it so that it works at the same speed as your altimeter board.

You will need to configure the module so that it works with your altimeter board.

Note that those modules usually come pre-set to work at 9600 bauds. However your altimeter board by default will work at 38400 bauds so will need to make sure that you are connecting at the correct speed.

CAUTION: make sure that you do not unplug the module while doing the following

Retrieving the module configuration

Plug the module in your Android device using a TTL cable and a USB adaptor that can work with micro USB or type C USB

You will need to select the bluetooth module that you have. If you do not know which one you have select the default one or any other one that is compatible. If you cannot use any of the module, send me the module references and/or pictures of the module.

Then click on [retrieve cfg] this will try to connect to the module using the default 38600 baud. If it fails try again, it will use various baudrates until it connects. It will then retrieve the module config so that you can change it

Saving the module configuration

Change the settings that you want to change such as the module name or speed. Make sure that you have selected a baudrate that will work with your board and click [Save cfg]. This will save the configuration. If any error occur while saving it will abort the save

Note that you will need to repeat the same steps for each bluetooth module that you want to configure
