Configuration du module 3DR

De manière à pouvoir faire de la télémétrie longue distance vous devez remplacer votre module bluetooth par un module télémétrie 3DR.

A noter que la batterie de votre appareil Android doit être suffisamment puissante pour alimenter le module 3DR. Envisager l'utilisation d'une batterie externe si vous avez un module puissant

Pour programmer votre module vous aurez besoins d'un cable TTL

un adaptateur USB (USBC ou micro USB suivant votre apareil Android )

un adaptateur jst et des cables jst

Vous devez configurer votre module de manière à ce qu'il puisse fonctionner avec votre altimètre.

A noter que ces modules sont en général préconfigurés à 57600 bauds. Cependant votre altimètre par défaut sera configuré à 38400 bauds vous devez donc vous assurer de vous connecter à la bonne vitesse.

A noter que les modules 3DR sont vendu par paire donc il sera nécessaire de configurer à la fois l'émetteur et le récepteur

ATTENTION: assurez vous de ne pas débrancher le module lorsque vous faites les opérations qui suivent

Charger la configuration du module

Branchez le module dans votre appareil Android en utilisant un câble OTG ou un adaptateur USB qui peut fonctionner avec micro USB ou USB type C.

Ensuite cliquez sur [charger cfg] , cela effectuera des tentatives de connexion au module à différentes vitesses jusqu'à ce qu'il se connecte. Puis la configuration de module sera chargée de manière à ce que vous puissiez la changer

Sauvegarder la configuration du module

Changez les paramètres que vous voulez changer. Assurez vous que vous avec choisi la vitesse qui fonctionnera avec votre carte et cliquez [Sauver cfg]. Cela sauvegardera la configuration. En cas d'erreur lors de la sauvegarde celle-ci sera annulée

A noter que vous devrez répéter la même opération pour chaque module
