Flashage du firmware

Vous devez en premier lieu choisir le firmware à flasher. Le firmware dépend de l'altimètre à flasher

Ensuite décidez si vous voulez directement flasher l'altimètre ou utiliser un firmware de récupération avant.

Utilisez un firmware de récupération si la configuration de l'altimètre n'est pas bonne et que celui ci ne peut plus se connecter.

Après avoir chargé un firmware de récupération vous devez flasher à nouveau votre altimètre.

Flasher votre altimètre avec le mauvais firmare aura pour conséquence d'avoir un altimètre qui ne fonctionne pas du tout ou qui ne fonctionne pas correctement

De manière à pouvoir flasher votre altimètre vous aurez besoin d'un câble USB et d'un adaptateur pour le brancher sur votre téléphone ou tablette.

un adaptateur USB (USBC ou micro USB en fonction de votre apareil Android )

et des cables jst

A ce jour, 3 familles d'altimètres existent, les altimètres à base d'ATMega328, les altimètres à base de STM32 et les altimètres à base d'un ESP32.

Flashage des altimètres à base d'ATMega328 (ex: AltiDuo, AltiServo, AltiMulti and AltiMultiV2)

- Branchez le cable

- Choisir le firmware approprié ou detectez le.

- Appuyez et relâcher le bouton reset de l'altimètre et appuyez sur le bouton [flash firmware]

Flashage des altimètres à base de STM32 (ex: AltiMultiSTM32 and AltGPS)

- Branchez le cable

- Choisissez le firmware approprié ou détectez le

- Déplacez le cavalier boot0 à droite

- Appuyez sur le bouton reset de l'altimètre

- Appuyez sur le bouton [flash firmware]

- Remettre le cavalier boot0 à gauche

Le firmware se charge, lorsque c'est terminé pressez le bouton reset

Flashage des altimètres à base d'ESP32 (ex: AltiMultiESP32)

- Branchez le cable

- Choisissez le firmware approprié ou détectez le

- Sur la carte pressez le bouton boot et maintenez le

- Sur la carte pressez le bouton reset et relacher le tout en maintenant le bouton boot

- Puis relacher le bouton boot

Le firmware devrait maintenant charger, si ce n'est pas le cas recommencez. Lorsque le chargement est complet appuyez sur le bouton reset

Flashage de l'altimètre TTGO

- Brancher le cable USBC

- Choisissez le firmware approprié pour l'altimètre TTGO

- press the [flash firmware] button

Firmware should now be loading, si ce n'est pas le cas recommencez. Lorsque le chargement est complet appuyez sur le bouton reset
