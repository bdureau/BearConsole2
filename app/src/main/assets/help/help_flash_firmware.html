Flash firmware

First you need to select the firmware. The firmware depend on the altimeter that you want to flash

Then decide if you want to directly flash your firmware or use a recovery firmware first

Use a recovery firmware if the config is wrong and the altimeter can no longer connect.

After using a recover firmware you will need to flash the firmware.

Note that flashing the wrong firmware will result in an altimeter not working at all or not working as expected

In order to be able to flash your altimeter you will need a USB TTL cable and an adapter to plug it to your phone.

a USB adapter (USBC or micro USB depending on your Android device)

some jst cables are required

Alternatively you can attempt to detect the firmware.

At the time of writing, 3 types of altimeters exists, the ATMega328 based altimeters, the STM32 based altimeters and the ESP32 based altimeters.

Flashing ATMega328 based altimeters (ie: AltiDuo, AltiServo, AltiMulti and AltiMultiV2)

- Plug the cable

- Select the correct firmware or detect it.

- Press and release the reset button on the altimeter and press the [flash firmware] button

Flashing STM32 based altimeters (ie: AltiMultiSTM32 and AltGPS)

- Plug the cable

- Select the correct firmware or detect it

- Move the boot0 jumper to the right

- Press the reset button

- Press the [flash firmware] button

- Move the boot0 jumper back to the left

press the [flash firmware] button

Firmware should be loading, when complete press the reset button

Flashing ESP32 based altimeters (ie: AltiMultiESP32 and AltiMultiESP32Accel)

- Plug the cable

- Select the correct firmware or detect it

- On the board Press the boot button and hold it

- On the board Press the reset button and release it while still pressing the boot button

- Then release the boot button

press the [flash firmware] button

Firmware should now be loading, if not try again. When complete press the reset button

Flashing TTGO altimeter

- Plug the USBC cable

- Select the correct Altimeter

- press the [flash firmware] button

Firmware should now be loading, if not try again. When complete press the reset button

Flashing The TTGO Mini altimeter or Simple altimeter

- Plug the USBC cable

- On the board Press the boot button and hold it

- On the board Press the reset button and release it while still pressing the boot button

- Select the correct Altimeter

Firmware should now be loading, if not try again. When complete press the reset button

Flashing TS-ESP32-S3 board altimeter

- Plug the USBC cable

- On the board Press the boot button and hold it

- On the board Press the reset button and release it while still holding the boot button

- Select the correct Altimeter

Firmware should now be loading, if not try again. When complete press the reset button

IMPORTANT NOTE: when flashing an altimeter do not disconnect the cable while flashing, this will corrupt the firmware and you will have to flash it again
