Bear Altimeter Console help file

This is a quick help file to use your Android Bear altimeter console. The Android console is a very convenient way to re-configure your altimeter when you are on the field.

Note that help is diffent on all screens, tt will display help on the current screen


First of all you need to make sure that your altimeter is compatible (ie: the correct firmware has been loaded).

Secondly make sure that the bluetooth module is connected on the altimeter.

Thirdly you need to go to your android device settings, go to the bluetooth configuration and search for the HC-05 or HC-07 bluetooth module. When you have find it enter the pairing code which by default is 1234.

!!! CAUTION !!!!

You might want to clearly name each bluetooth device paired in your phone to make sure that you configure the correct altimeter.

 

Then turn on the altimeter and wait until it beeps the continuity.

Then start the application on your Android device, click on the show paired devices and finally select the HC-05 or HC-06 device. It now attempt to connect to the altimeter.

If it says connected, you are now in and can use the Console to configure your altimeter or read the flights. If it cannot connect then make sure that you have the correct bluetooth device selected or that the altimeter is on.

 

Using the altimeter Console

Altimeter main screen

On the main screen you have a number of buttons, let's go and describe each one.

[Altimeter Setting]

Read the altimeter setting and allow you to modify them.

 

[Read Flights]

Read all the flights that have been recorded and plot them.

 

[Telemetry]

Go to the altimeter telemetry.  

 

[Continuity ON/OFF]

Turn the altimeter continuity on and off.  

 

[Reset Settings]

Give the ability to restore to factory settings the altimeter config. It also allow to clear all flights or just the last one. 

 

[State]

Go to the altimeter state screen. This will display some info about the altimeter (depending on the model) such as output continuity, memory usage, current altitude etc ..  

 

[Flash Firmware]

Allow to flash the firmware using a USB cable. This is available for Atmega 328, STM32 or ESP32 based board

 

[Connect/Disconnect]

Disconnect from the altimeter

 

Altimeter main screen menu

From the menu you can access this help file, an about screen that will give you informations about the console and its version and finally you have the option to customize the look and feel of this application.

 

 

Supported altimeters for this console are:

- Altiduo (large version with an Atmega328)

- AltiMulti V1

- AltiMulti V2

- AltiServo

- AltiGPS

- AltiMulti SMT32

- AltiMulti ESP32
